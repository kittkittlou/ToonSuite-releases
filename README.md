# ToonSuite

Downloads for the ToonSuite apps: tools for multi-tooners on Toontown Rewritten and Corporate Clash.

| App | What it does | Platforms |
|---|---|---|
| **Tunetoon** | Multi-account launcher: log in, patch, and launch your whole roster from one place | Windows, macOS |
| **Multicontroller** | Control many game windows with one keyboard and mouse | Windows |

Head to [**Releases**](../../releases) and grab the newest release for the app you want (Tunetoon releases are tagged `tunetoon-v…`, Multicontroller releases `multicontroller-v…`).

## Install

### Windows (both apps)
Download the `.exe` and put it in any folder you like. It's a portable app with no installer.
- **Tunetoon** keeps its settings in `%LocalAppData%\Tunetoon`.
- **Multicontroller** keeps its settings next to the exe, so keep it in its own folder.

### macOS (Tunetoon)
1. Download the zip for your Mac: `Tunetoon-macos-arm64.zip` (Apple Silicon) or `Tunetoon-macos-x64.zip` (Intel).
2. Unzip and drag `Tunetoon.app` to Applications (or anywhere).
3. **First launch:** right-click (Control-click) the app → **Open** → **Open**. macOS shows a warning because the app isn't notarized with Apple; this only happens once.

## Updates

Both apps check for updates when they start and can install updates from inside the app (About → Check for updates), so there's no need to come back here. In-app updates on macOS don't retrigger the first-launch warning.

## Verifying downloads

Each release includes a `SHA256SUMS.txt` with the checksums of every asset.
